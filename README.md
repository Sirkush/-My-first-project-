
# Student Performance Statistical Modeling

##  Project Overview
This project applies statistical modeling techniques to analyze and predict student academic performance. The goal is to identify key factors that influence GPA and overall academic success.

## Objectives
- Analyze student performance data
- Identify top predictive factors
- Build regression models
- Interpret statistical results using ANOVA and correlation analysis

## Methods Used
- Multiple Linear Regression
- ANOVA
- Correlation Analysis
- Exploratory Data Analysis (EDA)

##  Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

##  Key Insight
The analysis reveals the most influential variables affecting student performance and provides data-driven recommendations.
